# WeatherAppDroid

Предоставляет следующие данные о погоде для любых географических координат:
Текущая погода
Минутный прогноз погоды на 1 час
Почасовой прогноз погоды на 48 часов
Ежедневный прогноз погоды на 8 дней
Национальные погодные оповещения
Исторические данные о погоде за более чем 40 лет (с 1 января 1979 года)
Примечание :
Единый ключ API на have


Шаги :
Сначала вам нужно создать на нем учетную запись.

Затем сгенерируйте уникальный API-ключ, чтобы получить все данные из файла JSON.

Вставьте свой API-ключ в LocationCord.java файл как

публичная конечная статическая строка API_KEY = "81a26c8f0de407b94623e9f43e825679";



Provides the following weather data for any geographical coordinates:
Current weather
Minute forecast for 1 hour
Hourly forecast for 48 hours
Daily forecast for 8 days
National weather alerts
Historical weather data for 40+ years back (since January 1, 1979)
Note :
Single API key on have


Steps :
First, you have to create a account on it.

Then, generate a a unique API key to get all the data from the JSON file.

Paste you API KEY in LocationCord.java file as

public final static String API_KEY = "81a26c8f0de407b94623e9f43e825679";
